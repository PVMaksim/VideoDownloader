# CLAUDE.md — VideoGrab Backend

## Стек
- Язык: Python 3.12
- Фреймворк: FastAPI, Uvicorn
- БД: PostgreSQL 16 (asyncpg), Redis 7 (кеширование/очереди)
- Инфраструктура: Docker, Docker Compose, GitHub Actions (планируется)
- Внешние сервисы: Resend (email), yt-dlp (скачивание видео)

## Архитектура
Бэкенд для сервиса скачивания видео. Основные модули:
- `src/auth/` — аутентификация (JWT), регистрация, верификация email
- `src/downloads/` — логика скачивания видео через yt-dlp, управление задачами
- `src/db/` — модели SQLAlchemy, миграции Alembic
- `src/billing/` — (заглушка) платёжная логика
- `src/worker.py` — Celery worker для фоновых задач

Поток запроса: `POST /auth/register` → создание пользователя → отправка письма (Resend) → верификация → доступ к `/downloads`.

## Правила написания кода
- Все функции должны иметь docstrings (на английском)
- Комментарии к бизнес-логике: на русском
- Никаких «магических чисел» — только именованные константы из `config.py`
- Всегда обрабатывать исключения: `try/except` с логированием
- Модульная структура: один файл = одна ответственность
- Асинхронность: `async/await` для всех I/O-операций

- Документировать все публичные интерфейсы (FastAPI routers, Pydantic schemas)
