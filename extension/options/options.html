Бэкенд сервер

Адрес твоего VPS с запущенным VideoGrab backend
Значение API_KEY из файла .env на сервере
Как запустить бэкенд на VPS:
1. Скопируй папку backend/ на VPS
2. Создай .env из .env.example, задай API_KEY
3. Запусти: docker-compose up -d
4. Настрой Nginx + HTTPS на порт 8000
5. Вставь URL и ключ сюда
