- [ ] Нет обработки случая когда видео на странице не одно (несколько уроков)

### Принятые решения
- **webRequest вместо fetch/XHR перехвата** — GetCourse использует `lockdown-install.js` (SES), который откатывает изменения в `window.fetch`. webRequest работает на уровне браузера и неуязвим.
- **Два content script'а** — MV3 не позволяет одному скрипту работать в обоих world одновременно.
- **Бэкенд на VPS, не локально** — HLS состоит из сотен `.ts` сегментов, ffmpeg нужен для мержа. В браузере это невозможно.
- **yt-dlp вместо самописного загрузчика** — поддерживает 1000+ платформ, регулярно обновляется.
